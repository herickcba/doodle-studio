Instalar no PowerPoint

São dois componentes — instale os que quiser. Os arquivos vão para a pasta Downloads.

Faixa “CBA Studio”tipografia · entrelinha · âncora · cantos arredondados · cores
⬇ Add-in (.ppam) ⬇ Instalador (Mac)
  1. Baixe os dois arquivos acima (vão para Downloads).
  2. Abra o instalador. Se o Mac bloquear, clique com o botão direito → Abrir.
  3. Reabra o PowerPoint → Habilitar Macros. A aba CBA Studio aparece.
Extensãopainel Doodle + ImaGen
⬇ Instalador da extensão (Mac)
  1. Abra o instalador (botão direito → Abrir se for bloqueado).
  2. No PowerPoint: Inserir → Suplementos → Meus Suplementos → CBA Studio.
Faixa “CBA Studio”INSTALADOR BETA
⬇ Add-in (.ppam) ⬇ Instalador (.bat)
  1. Baixe os dois arquivos para a pasta Downloads.
  2. Rode o instalar-faixa-windows.bat.
  3. Abra o PowerPoint → Habilitar Macros. A aba CBA Studio aparece.

Manual (se a aba não aparecer): Arquivo → Opções → Suplementos → em “Gerenciar”, escolha Suplementos do PowerPoint → Ir… → Adicionar Novo… → selecione o .ppam.

Extensãopainel Doodle + ImaGen
⬇ manifest.xml
  1. No PowerPoint: Inserir → Suplementos → Meus Suplementos.
  2. Carregar Meu Suplemento (Upload My Add-in) → escolha o manifest.xml.
