Doodle Studio para PowerPoint

Suplemento (add-in) de desenho à mão direto nos seus slides

Desenhe doodles vetoriais (traço de giz, suavização, linha reta) e insira direto no slide como imagem transparente, posicionada onde você desenhou. Roda no PowerPoint do Mac, Windows e Web.

Baixar o suplemento Arquivo doodle-studio.xml (manifesto). Siga as instruções abaixo para instalar.

Como instalar

🍎 PowerPoint no Mac 1 clique

  1. Baixe o instalador (Mac) e abra o arquivo .zip.
  2. No instalar-mac.command, clique com o botão direito → Abrir (na primeira vez o Mac pede confirmação; clique em Abrir).
  3. Reabra o PowerPoint: Inserir → Add-ins, role até Developer Add-ins e clique em Doodle Studio.

Prefere o Terminal? Cole esta linha e tecle Enter:

mkdir -p ~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef && curl -fsSL https://doodle-studio-app.vercel.app/manifest.xml -o ~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef/doodle-studio.xml && osascript -e 'tell application "Microsoft PowerPoint" to quit'

🌐 PowerPoint na Web mais fácil

  1. Baixe o doodle-studio.xml.
  2. Em powerpoint.office.com, abra uma apresentação.
  3. Inserir → Suplementos → Carregar Meu Suplemento (Upload My Add-in) e selecione o arquivo.

🪟 PowerPoint no Windows desktop

O Windows desktop não tem cópia simples (o "Carregar Meu Suplemento" só existe na Web). Use o catálogo de pasta confiável, uma única vez:

  1. Baixe o doodle-studio.xml e coloque numa pasta, ex.: C:\DoodleStudio.
  2. Botão direito na pasta: Propriedades → Compartilhamento → Compartilhar. Anote o caminho de rede (ex.: \\SEU-PC\DoodleStudio).
  3. No PowerPoint: Arquivo → Opções → Central de Confiabilidade → Configurações da Central de Confiabilidade → Catálogos de Suplementos Confiáveis. Cole o caminho, clique Adicionar catálogo, marque Mostrar no Menu e OK.
  4. Feche e reabra o PowerPoint.
  5. Início → Add-ins (ou Inserir → Suplementos) → aba Pasta Compartilhada → Doodle Studio.

Guia oficial: catálogo de pasta compartilhada.

🚀 Instalar para um time inteiro, sem fricção

Para vários usuários instalarem com zero passos, o caminho certo é Implantação Centralizada (um admin do Microsoft 365 publica e o suplemento aparece para todos automaticamente) ou a AppSource (loja do Office, instala com 1 clique). Os métodos acima são para uso individual.

Como usar

  1. Com o painel aberto, escolha a cor e desenhe no canvas. Shift = linha reta.
  2. Para ver o slide como referência: copie o slide (clique nele → Cmd+C) e clique “Fundo do slide” (ou cole com Cmd+V).
  3. Clique “Inserir no slide” — o desenho entra como imagem no lugar onde foi desenhado.
  4. Quer mais espaço? “Tela grande para desenhar” abre uma janela ampla com o slide ao fundo.

Observação: o desenho é inserido como imagem (não forma editável), por limitação da API do Office.
