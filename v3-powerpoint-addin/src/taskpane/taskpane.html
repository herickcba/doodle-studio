Entrelinha — macro (configurar 1×)

A API do PowerPoint não define entrelinha por código. Aplique o estilo acima e rode BG_AplicarEntrelinha — ela lê o tamanho e acerta a entrelinha (0.8×/0.9×…). BG_InserirBloco cria uma caixa já 100% no estilo.

Baixar .bas
  1. Option+F11 → Inserir › Módulo → cole o código.
  2. Selecione o texto/objeto → Option+F8 → BG_AplicarEntrelinha (ou adicione ao Quick Access Toolbar para 1 clique).
Alternativa sem macro: insira a referência e use o Pincel de Formatação/copiar — também carrega a entrelinha.
